Ionrift Resonance

Automated audio triggers for Foundry VTT. Sound effects fire automatically from combat workflows, mechanics, and chat messages.

Supports Daggerheart (native) and DnD 5e (via Midi-QOL). Works with local sound files (WAV/MP3) out of the box — no paid accounts required. Optionally integrates with Syrinscape for cloud-hosted audio.

✨ What's New in v2.2.0

  • Configurable Sound Timing — Tune the delay between impact and vocal sounds, AoE stagger, spell audio bonus, and more. All exposed in the Orchestrator UI.
  • Polished Sound Picker — Distinct visual treatment for browsing vs. assigned sounds. Compact catalog rows with click-to-add, violet-tinted playlist section.
  • Clearer Actor/Item Config — "Pain / Hit" → "Vocal (Pain)", with Voice Tone dropdown only for PCs. Monster actors show monster-appropriate defaults.
  • Humanized UI — Picker titles, labels, and hints across all config screens now use clear, human-readable language.
  • 389 SFX Pack sounds — Local sound files covering combat, weapons, monsters, spells, and stingers. Metadata stripped clean.

⚔️ Combat Automation

Resonance listens to your game system's combat hooks and plays context-aware sounds:

  • Weapon swings — Sword slash, bow draw, spell cast (auto-detected from item type)
  • Hit / Miss / Crit / Fumble — Distinct audio cues per outcome
  • Pain & Death vocals — Per-PC identity (Deep/Bright), per-monster type (Undead, Beast, Dragon…)
  • AoE mitigation — Single impact + staggered vocal chorus instead of 20 overlapping sounds

🎧 Sound Sources

  • Local Files — Drop WAV/MP3 files anywhere in your Foundry data folder. No accounts needed.
  • Ionrift SFX Pack — 389 sounds included. Select "Ionrift SFX Pack" preset to activate.
  • Syrinscape (optional) — Cloud-hosted sound library via auth token.

🔧 Configuration

  • Attunement Protocol — Guided first-run wizard for token setup and preset selection
  • Resonance Calibration — Full sound binding editor with 4 tiers: Core Events → Categories → Monsters → Campaign Overrides
  • Per-Actor & Per-Item Overrides — 🎵 Sounds button on every sheet. Custom attack, pain, death sounds per character or weapon.
  • Orchestrator — Budget windows prevent sound spam. Configurable timing offsets for layered sounds.
  • Import / Export — Full JSON backup and restore of all sound bindings

🎲 Supported Systems

Daggerheart Native support. Fear Tracker, Duality Dice, Hope/Stress/Armor mechanics, Domain sound resolution.
DnD 5e Via Midi-QOL. Attack → Hit/Miss → Damage → Pain/Death chain. Spell school detection. Creature classification.
